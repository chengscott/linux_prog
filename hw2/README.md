# Process Finder

## Usage

`make` builds a binary `hw2`

## help

```shell
hw2 - Process Finder
usage: ./hw2 [-a] [-x] [-pqrs] [-t]

-a Show processes from all users
-x Show processes without associated terminal
-p Sort by pid (default)
-q Sort by ppid
-r Sort by pgid
-s Sort by sid
-t Show processes in tree view
```
